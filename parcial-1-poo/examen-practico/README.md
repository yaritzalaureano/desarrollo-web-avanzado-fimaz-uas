EXAMEN
Mini-sistema de usuarios desarrollado en PHP con Programación Orientada a Objetos (POO).
Incluye validación de correo con excepciones, herencia entre clases (Usuario, Admin, Alumno) y visualización de los usuarios en una tabla HTML.

Funciones principales

Clase Usuario con validación de correo.

Clase Admin que retorna el rol Administrador.

Clase Alumno con atributo matricula y rol Alumno.

Uso de try/catch para manejar correos inválidos.
