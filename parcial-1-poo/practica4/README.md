Práctica 4 – Integración POO + Herencia + Validaciones + Excepciones
Objetivo de la práctica

Implementar un sistema básico utilizando Programación Orientada a Objetos en PHP que incluya herencia, validación de datos y manejo de excepciones.
El sistema crea diferentes tipos de usuarios (Administrador, Alumno e Invitado) y muestra la información en una tabla HTML.

Requisitos

Para ejecutar la práctica se requiere:

PHP 8 o superior

Servidor local con XAMPP

Navegador web

Editor de código (por ejemplo Visual Studio Code)

Ruta de ejecución en el navegador

El proyecto debe colocarse dentro de la carpeta htdocs de XAMPP.

Luego abrir en el navegador la siguiente ruta:

http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/index.php

Evidencia esperada

Al ejecutar el programa se debe observar:

Una tabla HTML que muestre los usuarios creados (Administrador, Alumno e Invitado).

Las columnas: Nombre, Correo, Rol, Matrícula y Empresa.

En las columnas que no aplican se muestra el símbolo —.

Un mensaje de error controlado cuando se intenta crear un usuario con un correo inválido.
